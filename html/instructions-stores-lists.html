Tiendas Especiales

Para incluir sólo ciertas tiendas o excluir algunas tiendas, hay que seguir los siguientes pasos:

  1. Definir listas de tiendas: En la segunda pestaña del input puedes poner una lista de tiendas por columna. El nombre de cada columna es el identificador de la lista y tú lo defines. Las listas pueden ser de longitudes diferentes.
  2. Incluir tiendas: Para que una promoción impacte únicamente en ciertas tiendas, basta con que en el campo "" del input le pongas el nombre de la lista que quieres utilizar. Nota que si usas esta funcionalidad, únicamente se incluirán las tiendas del formato especificado que aparezcan en la lista, y se ignorará la lista de exclusión.
  3. Excluir tiendas: Para excluir ciertas tiendas de una promoción, solamente tienes que especificar la lista de tiendas que quieres excluir en el campo "" de la promoción.
  4. Notas:
    1. Si no quieres incluir ni excluir tiendas específicas, sólo deja en blanco las columnas del input.
    2. Es necesario repetir el nombre de la lista para todos los renglones dentro de una misma promoción, pero puede variar entre promociones.
    3. Si intentas incluir y excluir tiendas al mismo tiempo, se ignorará la lista de exclusión.
    4. La pestaña con las listas debe ser la segunda del archivo de entrada.
    5. Puedes dejar listas que no se utilicen en ninguna promoción, pero si el nombre de una lista aparece en una promoción, es indispensable que la lista exista en la segunda pestaña.
