Muebles

Las columnas de muebles permiten calcular una aproximación del número máximo de piezas que puede cargar una exhibición, incluyendo exhibiciones compartidas entre varios artículos.

Después de que seleccionas el mueble, se usa un algoritmo que verifica su existencia en cada una de las tiendas del formato. Una vez encontrado el mueble, se calculan las piezas máximas con base en las dimensiones del mueble, de los artículos y su participación de venta.

  • Mueble deseado: Es el mueble asignado en la GDM.
    • Uso: Las únicas opciones válidas son: __shelves__.
  • Mueble predeterminado: En caso de que en una tienda no se encuentre el mueble deseado, entonces se buscará este mueble como segunda opción para el cálculo.
    • Uso: Puedes elegir entre las mismas opciones que el mueble deseado, o puedes definir el número máximo de piezas a cargar entre todos los artículos de la exhibición por tienda.
    • Características especiales:
      • Si en una tienda no se encuentra el mueble predeterminado para una tienda, entonces se usará un promedio de las medidas.
      • Si ingresaste el número de piezas, en caso de que para una tienda no se encuentre el mueble deseado, el número que ingresaste pasará directamente a ser el máximo de piezas en la exhibición (sumando todos los artículos).
